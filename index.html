How EDIZz Works

Step 1: Onboarding

Choose Student or Teacher and complete basic details.

Step 2: Verification

Teachers submit a live 30-second verification video demonstrating their skill.

Step 3: Profile Setup

Teacher adds field of teaching, experience, and description for students.

Secure Connect

Student uses teacher cards, pays to unlock chat via Escrow, then starts learning.

Why Choose EDIZz?

🎙️ Smart Voice Matchmaking

Users speak their goals. Our AI extracts the exact skill, timeframe, and intent to find the perfect local match.

🔒 Escrow Payment Security

Learners pay upfront, but funds are only released to the educator after the first lesson is completed.

✅ Quality Verification

Tutors must upload a 30-second skill demonstration to earn a "Verified" badge, guaranteeing quality and safety.

🌍 The Impact

EDIZz is bridging the trust and friction gap in local skill-sharing. It empowers GenZ to become self-employed and builds stronger local community bonds.

🛡️ Privacy First

We use radius-based location matching instead of exact pins to ensure user privacy. In-app reviews and progression badges keep the ecosystem trusted and secure.

Technical Backbone

Frontend: Mobile-responsive Web App for frictionless, cross-platform access.
Backend API: Python (FastAPI) for high-speed, asynchronous API routing.
Voice AI: Deepgram API for fast Speech-to-Text, and Gemini/OpenAI for intent extraction.
Matching Engine: Pandas DataFrames utilized for instant, vectorized in-memory matching with zero-latency geographic filtering.
